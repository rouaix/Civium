Principes du projet

  • Protocole ouvert, aucune entreprise ne le possède
  • Pas de données centralisées, les contributions restent dans le dépôt public
  • Si Civium échoue ou déçoit, n'importe qui peut forker le protocole
  • La spec fait foi, tout changement passe par le README et le ROADMAP
